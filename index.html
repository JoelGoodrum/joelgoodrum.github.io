CarApp

CarApp is a Java program which simulates a mobile app where the users get o navigate a list of cars in order to purchase a one. When the user clicks on the type of car, the page shows the options available for that type. The user then can click on a specific model. After the user clicked on a model, the details of the car will show up. All the elements in this program were programmed in Java.

GraphingCalc

GraphingCalc is a simple graphing calculator that was written in Java. In this program, one can write equations in the format y = x. To write the functions, one must go to line 53 in the source code and change the values that effect x. A set amount of pixel was set to equal the distance of 1 on both x and y axis. The program begins ploting values from the left corner of the screen and prints for each new value after shifting 1 unit on the graph.
