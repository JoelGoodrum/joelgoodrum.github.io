Projects

Graphing Calculator

In this program, one can write equations in the format y = x. The program calculates the value of y for each input of x and plots it on the screen. Each row has total of 14-unit squares. Each square has about 25 pixels. The program converts unit squares into pixels and plots individual points for each pixel. The program's precision goes a little beyond one pixel per dot. it calculates precisely 14000 points in less than half a second. This program was written with Java using the JavaFX library.

Hospital Simulator

Hospital Simulator is a program which simulates the check in and check out of patients at a hospital. The user can submit as many patients as needed to a priority queue. There are only three surgery rooms in the hospital. The software chooses the next patient based on first com first serve and by priority. This program was written in Java using the JavaFX library.

iPhone Lock Screen

This project, one can see the current time on the lock screen of an iPhone. I designed the phone using html and CSS. I used JavaScript to output the date and time from the computer.

CarApp

CarApp is a program where the user can navigate a list of cars. When the user clicks on the type of car, the page shows the options available for that type. The user then can click on a specific model. After the user clicked on a model, the details of the car will show up. CarApp runs only on computers. Everything was coded in pure Java.

WizardRPG

In WizardRPG, one gets to strategically fight mystical creatures. All players have health and magic points. The player can attack or defend against his opponent. The opponent also has the options of defending or attacking. The player must bring the opponents magic or health points to zero in order to win. This game was programmed in html, CSS and JavaScript.
